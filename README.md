# Fund

Fund是一个方便在终端环境下追踪基金趋势的小工具<br>
数据来源是天天基金网

<img src="https://github.com/xuchdeid/fund/blob/master/screenshot/screenshot1.png?raw=true" width = "400" alt="screenshot" align=center />

## 如何使用

```sh
npm install -g ./fund

// 输入要跟踪的基金代码
fund 001549 000961 241001 164906 001593 001632

// 默认60s刷新一次
// --delay 30 可以30s刷新一次
```
